Help us translate SABnzbd in your language!
Thank you for your interest! Below you can find the link to our online translation system, hosted by Transifex.
To start translating, you will need to create an account with Transifex by clicking `Join this project` on the right. You can list `Localization` as your Department and `Translator` as your Job Title.
You should now be able to click Join Team. If not, click the link below again after registering to find us again.
For more information on the translation interface, you can read the [Transifex documentation](https://help.transifex.com/en/).
NOTE Confirming your email address is required to start translating.
NOTE The username and email address you use will be automatically added to the SABnzbd source files if you contribute translations.
NOTE If you want to be able to Review or Coordinate translations, please contact us on the [forums](https://forums.sabnzbd.org/) so we can grant you additional permissions.
