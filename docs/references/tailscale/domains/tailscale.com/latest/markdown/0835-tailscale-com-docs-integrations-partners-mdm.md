# Tailscale MDM integration partners
Last validated: Dec 2, 2025
MDM integrations are available for [the Standard, Premium, and Enterprise plans](/pricing).
Mobile device management (MDM) solutions such as those listed below let you apply a set of [system policies](/docs/features/tailscale-system-policies) across all the devices in your fleet that are running Tailscale. System policies can be used to customize the behavior of the Tailscale client to match the policies and use cases of your organization.
* [Deploy Tailscale with Google Workspace](/docs/integrations/mdm/google-workspace)
* [Deploy Tailscale with Jamf](/docs/integrations/mdm/jamf)
* [Deploy Tailscale with JumpCloud](/docs/integrations/mdm/jumpcloud)
* [Deploy Tailscale with Iru](/docs/integrations/mdm/kandji) (formerly Kandji)
* [Deploy Tailscale with Microsoft Intune](/docs/integrations/mdm/microsoft-intune)
* [Deploy Tailscale with SimpleMDM](/docs/integrations/mdm/simplemdm)
* [Deploy Tailscale with TinyMDM](/docs/integrations/mdm/tinymdm)
While the list above includes MDM solutions we have configured, tested, and verified, Tailscale will work with other MDM solutions.
The Tailscale client can be both deployed and configured using MDM solutions. We support MDM functionality on [Android](/docs/integrations/mdm/android), [iOS/tvOS](/docs/integrations/mdm/ios), [macOS](/docs/integrations/mdm/mac), and [Windows](/docs/integrations/mdm/windows-mdm).
