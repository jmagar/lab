# Set up LLM providers
Last validated: Apr 15, 2026
Aperture by Tailscale is currently [in beta](/docs/reference/tailscale-release-stages#beta).
Each provider requires API credentials and a configuration entry in the [Aperture configuration](/docs/aperture/configuration). After you configure a provider, any [LLM client set up to use Aperture](/docs/aperture/use-your-tools) can access models from that provider automatically. Refer to the [provider compatibility reference](/docs/aperture/provider-compatibility) for the full matrix of supported providers, API formats, and compatibility flags.
