### Secure Networking with Tailscale and AWS
### Agent-to-agent networking
Establish direct agent-to-agent connections and use Tailscale SSH to connect to EC2 instances without messy key management.
### IP-based routing
Use Tailscale subnet routers to easily connect managed services such as Amazon's Relational Database Service (AWS RDS) and Amazon RedShift when agents can't be installed.
### S3 for recording persistence
Use Tailscale SSH session recording to persist recordings to an Amazon S3 bucket. Reduce operational concerns such as available storage space, durability, and access controls.
