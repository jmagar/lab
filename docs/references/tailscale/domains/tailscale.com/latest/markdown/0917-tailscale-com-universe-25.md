### Tailscale \<\> GitHub
### Deploy securely from GitHub Actions, without exposing infrastructure
Tailscale’s official GitHub Action lets your workflows securely connect to private resources - databases, test runners, staging servers, internal APIs - without opening a single port to the public internet.
[
Learn more
](https://tailscale.com/blog/private-connections-for-github-actions)
### GitOps for Tailscale Access Policies
Manage access controls the same way you manage code. With GitHub Actions, you can review, test, and automatically deploy ACL changes to your Tailscale network using Tailscale’s GitOps integration.
[
Learn more
](https://tailscale.com/kb/1306/gitops-acls-github)
### Keep Secrets Secure with GitHub Secret Scanning
Avoid leaking Tailscale auth keys or OAuth secrets in your repos. If a secret is detected, Tailscale is alerted, automatically revokes it, and keeps your network protected, reducing the risk of CI/CD compromise.
[
Learn more
](https://tailscale.com/blog/github-secret-scanning)
### Private Resource Access in GitHub Codespaces
Install Tailscale in GitHub Codespaces to connect your ephemeral development environments to private services across your network. Everything remains private, protected, and scoped to your team.
[
Learn more
](https://tailscale.com/kb/1160/github-codespaces)
