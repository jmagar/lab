# Control D, meet Tailscale
We’re excited to announce an integration with [Control D](https://controld.com/), a customizable DNS service. With Control D and Tailscale, every device on your tailnet can be protected from internet threats, unwanted and malicious content, or ads.
To use Control D with Tailscale, create a Control D account, and then set Control D as your tailnet’s global nameserver by copying one of the Control D profile IP addresses into your tailnet’s DNS settings. Select “Override local DNS” to force your devices to use Control D.
If you’d like to get more granular, Control D is aware of individual Tailscale devices, letting you analyze traffic patterns and assess risk on each device separately. There are a wealth of [features](https://docs.controld.com/docs/business-features) you can enable with Control D and Tailscale.
For more information, see [our Control D documentation](https://tailscale.com/kb/1403/control-d).
