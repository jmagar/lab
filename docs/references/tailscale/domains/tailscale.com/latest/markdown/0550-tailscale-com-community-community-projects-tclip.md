# tclip
tclip is a private pastebin solution that allows secure text sharing between network users without exposure to the public internet, ideal for sensitive information sharing.
This pastebin alternative is self-hostable or can be easily deployed to fly.io. It incorporates tsnet, the Go library that enables service to natively use Tailscale features, and can be configured to expose select elements to the public internet using Tailscale Funnel.
