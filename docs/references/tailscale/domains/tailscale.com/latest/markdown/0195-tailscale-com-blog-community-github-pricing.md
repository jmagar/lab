# Free Tailscale Plan for Open Source GitHub Organizations
Tailscale loves open source. We know that it can be tough to develop a project in the open, and collaborate with individuals and organizations around the world.
We’re excited to announce that Tailscale is free for GitHub organizations using Tailscale for open source projects. And given [Tailscale is good at, well, making connections](/blog/how-nat-traversal-works/), friends and family who coordinate using GitHub organization accounts can also benefit from this free plan.
### New Community on GitHub plan
If you have an open source project with an [OSI license](https://opensource.org/licenses) that uses a GitHub organization, Tailscale is free for your organization. (At this point in time, we’re only offering this for GitHub orgs due to how we manage users—but if you use another identity provider, please [let us know](https://github.com/tailscale/tailscale/labels/identity).)
With the new Community on GitHub plan, you get:
* Up to 25 users
* 5 devices per user
* 1 subnet router
* 2 admin users
* 2 unique users in ACL policy
* Community support
We hope to be able to add more functionality over time as Tailscale continues to improve. Need more devices or having trouble with these restrictions? [Let us know](<mailto:info@tailscale.com?subject=Open source limits>).
This plan is also available to families and friends. Connect to your dad’s [photo server](/kb/1131/synology/), provide feedback on your daughter-in-law’s [new app](/kb/1113/aws-lambda/), and check in on your neighbor’s [shared driveway webcam](/kb/1076/dogcam/).
Tailscale also offers a 50% discount off of list prices for charities, not-for-profit organizations, and educational institutions. [Contact us](<mailto:info@tailscale.com?subject=Not for profit discount>) to get this discount.
### Personal will always be free
After [we introduced new pricing last month](/blog/2021-06-new-pricing/), we heard you—you were afraid we would change our free Personal tier. That’s not at all the plan.
The Personal plan for using Tailscale for your personal use on your own devices, will always be free. We remember how hard it is to connect to [our Pi-hole](/kb/1114/pi-hole/), or [our Minecraft server](/kb/1137/minecraft/), or [our dogcam](/kb/1076/dogcam/), and don’t wish that on anyone. It’s not going anywhere.
### Get started with Community on GitHub
To upgrade your GitHub organization to the new free Community on GitHub plan, select the new plan in the [Billing settings of the admin console](https://login.tailscale.com/admin/settings/billing). We’re proud to support open source.
