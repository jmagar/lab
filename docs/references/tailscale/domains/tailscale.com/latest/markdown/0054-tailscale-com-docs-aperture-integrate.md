# Integrate with external tools
Last validated: Apr 9, 2026
Aperture by Tailscale is currently [in beta](/docs/reference/tailscale-release-stages#beta).
Aperture supports integration with external tools for authorization, observability, and custom automation. Use hooks in the [Aperture configuration](/docs/aperture/configuration) to send event data to third-party services when requests match grant conditions. For built-in observability features, refer to [Observe and export AI usage](/docs/aperture/observe-and-export).
