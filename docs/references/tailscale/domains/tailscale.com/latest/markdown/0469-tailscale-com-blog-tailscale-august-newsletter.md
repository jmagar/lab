**From the team**
[Sync Azure AD users and groups to Tailscale](/blog/sync-azuread-groups/)
When you use Tailscale with Azure AD, your users login and authenticate with your existing identity provider (IdP), which means that if you remove a user from Azure AD, they’ll no longer be able to log into Tailscale.
[Introducing Machine Explorer, now in Tailscale’s VS Code extension](/blog/machine-explorer-vscode-extension/)
Tailscale for VS Code just got a major upgrade, bringing you the ability to explore, edit, and transfer the files on any of the nodes in your tailnet that you can reach through Tailscale SSH
[Tailnet lock is now available in beta](/blog/tailnet-lock-beta/)
Tailnet lock lets you verify that nodes can only be added to your tailnet when those nodes are signed by trusted nodes in your tailnet, which you manage.
[tclip: A pastebin for your tailnet](https://tailscale.dev/blog/tclip)
Check out the story behind our release of tclip, an open source tool for sharing snippets of text with other folks on your tailnet.
