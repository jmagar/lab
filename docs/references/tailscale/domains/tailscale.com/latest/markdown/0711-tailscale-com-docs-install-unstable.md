# Install unstable Tailscale clients
Last validated: Jan 7, 2025
Tailscale tests new features and fixes in unstable release clients before releasing updates
to the wider community. If you'd like to help test new features, you can download and install unstable clients
from the links below.
Features can be in various [release stages](/docs/reference/tailscale-release-stages). Beta features do not necessarily require an unstable release client to test.
Installing unstable builds are recommended for testing purposes only. While we strive to keep these builds reliable, they are experimental, and some are not signed for distribution.
## [Linux](#linux)
To install an unstable build of the Tailscale client on Linux, use one of the following methods.
[Packages site](/docs/install/unstable?tab=packages+site)[Tailscale CLI](/docs/install/unstable?tab=tailscale+cli)
1. Go to the Tailscale Packages site, [unstable](https://pkgs.tailscale.com/unstable) page.
2. Download the build for your distribution.
3. Follow the instructions for the distribution you are using to install the unstable build.
## [Windows](#windows)
To install an unstable build of the Tailscale client on Windows, use one of the following methods.
[Packages site](/docs/install/unstable?tab=packages+site)[Tailscale CLI](/docs/install/unstable?tab=tailscale+cli)
1. Go to the Tailscale Packages site, [unstable](https://pkgs.tailscale.com/unstable/#windows) page, Windows section.
2. Download the Windows version applicable to your version.
3. Install the build.
## [macOS](#macos)
To install an unstable build of the Tailscale client on macOS, use one of the following methods.
[Packages site](/docs/install/unstable?tab=packages+site)[Tailscale CLI](/docs/install/unstable?tab=tailscale+cli)[App Store](/docs/install/unstable?tab=app+store)
1. Go to the Tailscale Packages site, [unstable](https://pkgs.tailscale.com/unstable/#macos) page, macOS section.
2. Download the either the `.zip` or `.pkg` version of macOS build.
3. Install the build.
## [iOS](#ios)
To install an unstable build of the Tailscale client on iOS, join our [TestFlight unstable track](https://testflight.apple.com/join/tLcYLZJV).
To stop testing unstable versions:
1. Open the TestFlight app.
2. Select **Tailscale**.
3. Scroll to the bottom, then select **Stop Testing**.
## [Android](#android)
To install an unstable build of the Tailscale client on Android, join our [Play Store unstable track](https://play.google.com/apps/testing/com.tailscale.ipn).
## [tvOS](#tvos)
To install an unstable build of the Tailscale client on tvOS for an Apple TV, join our [TestFlight unstable track](https://testflight.apple.com/join/tLcYLZJV). This requires tvOS 17 or later.
To stop testing unstable versions:
1. Open the TestFlight app.
2. Select **Tailscale**.
3. Select the menu, then **Stop Testing**.
