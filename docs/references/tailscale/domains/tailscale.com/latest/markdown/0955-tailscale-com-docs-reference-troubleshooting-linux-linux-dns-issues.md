# Troubleshoot Linux DNS issues
Last validated: Feb 18, 2026
Tailscale attempts to interoperate with any Linux DNS configuration it finds already present. Unfortunately, some are not entirely amenable to cooperatively managing the host's DNS configuration. For information about troubleshooting, refer to [Configuring Linux DNS](/docs/reference/linux-dns).
