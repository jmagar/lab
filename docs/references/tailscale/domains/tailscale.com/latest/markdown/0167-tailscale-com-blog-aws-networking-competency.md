# Tailscale achieves AWS Networking Competency
We are delighted to announce that Tailscale has been awarded the [AWS Networking Competency](https://aws.amazon.com/networking/partner-solutions/?blog-posts-cards.sort-by=item.additionalFields.createdDate&amp;blog-posts-cards.sort-order=desc&amp;partner-case-studies-cards.sort-by=item.additionalFields.sortDate&amp;partner-case-studies-cards.sort-order=desc). This recognition underscores Tailscale's deep expertise and proven success in delivering seamless, secure networking solutions on AWS.
## [What this means for our customers](#what-this-means-for-our-customers)
Achieving the AWS Networking Competency highlights Tailscale's expertise in providing secure, reliable, and easy-to-use networking solutions for businesses leveraging AWS services. With Tailscale, customers can expect enhanced connectivity, simplified network management, and robust security for their cloud environments.
## [Key advantages of Tailscale on AWS](#key-advantages-of-tailscale-on-aws)
* **Unmatched security**: Tailscale's zero-trust networking model ensures that data remains protected and accessible only to authorized personnel, mitigating security risks.
* **Effortless management**: With Tailscale's intuitive interface and straightforward setup, managing your network becomes seamless, reducing operational overhead and complexity.
