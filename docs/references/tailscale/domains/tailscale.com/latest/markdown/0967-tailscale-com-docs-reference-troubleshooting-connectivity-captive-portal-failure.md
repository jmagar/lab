# Blocked by captive portal
Last validated: Mar 23, 2026
When you connect to a public network, you might be redirected to a [captive portal](https://en.wikipedia.org/wiki/Captive_portal) requiring you to authenticate or accept terms before accessing the internet. Refer to [Using Tailscale with captive portals](/docs/integrations/captive-portals) for troubleshooting guidance.
