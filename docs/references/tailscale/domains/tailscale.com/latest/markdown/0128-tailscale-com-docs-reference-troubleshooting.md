# Troubleshooting guide
Last validated: Jan 5, 2026
This topic contains various suggestions and tips to help troubleshoot common issues. If you have additional issues, [contact Tailscale Support](/contact/support).
## [Connectivity and network issues](#connectivity-and-network-issues)
* For basic device connectivity issues, refer to [Troubleshoot device connectivity](/docs/reference/troubleshooting/connectivity).
* For network configuration issues, refer to [Troubleshoot network configuration](/docs/reference/troubleshooting/network-configuration).
* For general network troubleshooting (that is, not specific to Tailscale), refer to [Basic network troubleshooting](/docs/reference/troubleshooting/basic-network-troubleshooting).
## [Issues related to a specific operating system](#issues-related-to-a-specific-operating-system)
For OS-specific Tailscale troubleshooting, refer to the following:
* [Troubleshooting guide for Apple](/docs/reference/troubleshooting/apple)
* [Troubleshooting guide for Linux](/docs/reference/troubleshooting/linux)
* [Troubleshooting guide for Windows](/docs/reference/troubleshooting/windows)
## [Mobile device issues](#mobile-device-issues)
For mobile device issues, refer to [Troubleshooting mobile devices](/docs/reference/troubleshooting/mobile).
## [Container issues](#container-issues)
For container issues, refer to [Troubleshooting containers](/docs/reference/troubleshooting/containers).
## [Cloud environment issues](#cloud-environment-issues)
For cloud environment issues, refer to [Troubleshooting cloud environments](/docs/reference/troubleshooting/cloud).
## [Additional resources](#additional-resources)
If you need to generate a bug report to help Tailscale Support debug an issue, follow the instructions at [Generate a bug report](/docs/account/bug-report).
If you want to enable debug mode for the Tailscale client menu to assist in troubleshooting, follow the instructions at [Debug menu and options](/docs/reference/debug-menu).
