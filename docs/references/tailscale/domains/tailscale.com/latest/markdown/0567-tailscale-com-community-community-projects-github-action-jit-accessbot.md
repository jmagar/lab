# GitHub Action JIT Accessbot
This GitHub Action allows Tailscale users to request and approve just-in-time access to resources on their tailnet.
The Action uses Tailscale's custom device posture attributes to grant instaneous and time-bound access to designated machines or services.
Note that custom device posture attributes are available for the Personal, Personal Plus, and Enterprise plans. In addition, this Action requires a feature flag be enabled by Tailscale Support before it can be used.
