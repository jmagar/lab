# Tailscale client flagged by antivirus software
Last validated: Mar 24, 2026
The Tailscale client can be falsely flagged by antivirus software, especially with new releases, because many antivirus engines use machine learning and other automated tools that can result in false positives. For recommended actions and to report concerns, refer to [Antivirus flagging of the Tailscale client](/docs/concepts/av-flagging).
