# Infrastructure as code
Last validated: Jan 28, 2025
Use Tailscale's integrations with Infrastructure as Code (IaC) platforms to deploy infrastructure programmatically to your Tailscale network (known as a tailnet). IaC tools let you manage Tailscale resources and settings, including tailnet policy configurations, DNS settings, device properties, and auth keys.
