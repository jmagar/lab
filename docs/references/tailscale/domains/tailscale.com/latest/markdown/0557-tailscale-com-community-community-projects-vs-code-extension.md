# VS Code extension
This VS Code extension brings an explorer view of machines on your tailnet to the text editor, and allows you to connect to those devices and modify or edit remote files over Tailscale SSH.
In addition, the extension allows you to expose a local port to the public internet with Tailscale Funnel, as a lightweight nGrok alternative right in your text editor.
