# Troubleshoot macOS system extension errors
Last validated: Feb 18, 2026
When launching Tailscale for the first time on your Mac, you may be presented with **System Extension Blocked** or **System Extension Approval Required** messages. If so, refer to [Authorizing the Tailscale system extension on macOS](/docs/concepts/macos-sysext) for next steps.
