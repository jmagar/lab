# Observe and export AI usage
Last validated: Apr 9, 2026
Aperture by Tailscale is currently [in beta](/docs/reference/tailscale-release-stages#beta).
Aperture captures telemetry for every LLM request, including token counts, model names, session grouping, and tool use data. You can access this data through the Aperture dashboard or export it for integration with external monitoring and compliance tools. For more information on what Aperture captures and how session tracking works, refer to [How Aperture works](/docs/aperture/how-aperture-works).
You can also connect Aperture with external services such as log aggregators and authorization engines. Refer to [Integrate with external tools](/docs/aperture/integrate) for available integrations.
