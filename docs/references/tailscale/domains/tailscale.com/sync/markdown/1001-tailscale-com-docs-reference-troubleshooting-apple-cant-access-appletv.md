# Troubleshoot Apple TV
Last validated: Feb 18, 2026
The most common reason an Apple TV cannot be accessed is because it has gone to sleep and is not configured as a home hub, which lets it remain connected to the network while in sleep mode. For information about configuring and verifying Apple TV as a home hub, refer to [Install Tailscale on an Apple TV](/docs/install/appletv#troubleshooting).
