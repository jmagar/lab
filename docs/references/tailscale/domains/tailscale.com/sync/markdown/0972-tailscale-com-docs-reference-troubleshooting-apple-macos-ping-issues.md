# Troubleshoot macOS ping issues
Last validated: Feb 18, 2026
macOS "stealth mode" will prevent macOS from responding to pings. For information about stealth mode, refer to Apple's [stealth mode](https://support.apple.com/guide/mac-help/use-stealth-mode-to-keep-your-mac-more-secure-mh17133/26/mac/26) topic.
