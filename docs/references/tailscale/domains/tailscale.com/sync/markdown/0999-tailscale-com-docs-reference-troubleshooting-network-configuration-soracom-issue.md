# Troubleshoot SoraCOM issues
Last validated: Mar 16, 2026
Tailscale can work on SoraCOM but not with the identity service provider's DNS servers. We recommend enabling the **Override DNS servers** toggle on the [DNS](https://login.tailscale.com/admin/dns) page of the admin console.
