# Tailscale ACL combiner
The Tailscale ACL combiner is a utility for combining fragments of Tailscale policy files into a single huJSON object.
Tailscale ACLs are ultimately stored as one huJSON object, but for some workflows—especially ones based on GitOps principles—it can be neater to maintain different components in separate files. The Tailscale ACL combiner allows you to track those files separately and combine them in a standardized manner for production.
