# Manage AI spending
Last validated: Apr 9, 2026
Aperture by Tailscale is currently [in beta](/docs/reference/tailscale-release-stages#beta).
Aperture tracks token usage and cost across all LLM requests. You can use quotas to set spending limits per user or team, monitor budget consumption, and refill balances when needed. Configuration options for quotas are set in the [Aperture configuration](/docs/aperture/configuration) and monitoring is available through the [Aperture dashboard](/docs/aperture/reference/dashboard).
