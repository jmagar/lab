# Mobile device batteries drain quickly
Last validated: Mar 20, 2026
Draining of mobile device batteries is a known issue ([#3363](https://github.com/tailscale/tailscale/issues/3363)). However, we are continuously making improvements. Battery drain is most commonly attributed to a device using an [exit node](/docs/features/exit-nodes) for all traffic.
