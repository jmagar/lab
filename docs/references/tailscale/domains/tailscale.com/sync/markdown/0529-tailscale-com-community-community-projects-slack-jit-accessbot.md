# Slack JIT Accessbot
The Tailscale Slack JIT Accessbot allows Tailscale users to request and approve just-in-time access to resources on their tailnet.
The bot uses Slack workflows and Tailscale's custom device posture attributes to grant instaneous and time-bound access to designated machines or services.
Note that custom device posture attributes are available for the Personal, Personal Plus, and Enterprise plans.
