# Free pricing plans and discounts
Last validated: Apr 8, 2026
Tailscale offers several free plans and discounted pricing options to support individuals, open source projects, and eligible organizations. For an overview of Tailscale's pricing plans, including paid plans, refer to our [Pricing](/pricing) page.
You need to be an [Owner, Admin, or Billing admin](/docs/reference/user-roles) to administer your Tailscale plan.
## [Personal plan](#personal-plan)
The Personal plan permits 6 free users in a single Tailscale network, known as a tailnet. You can also share devices with other users with [node sharing](/docs/features/sharing). For more information on what is included in the Personal plan, refer to our [Pricing](/pricing) page.
## [Community on GitHub](#community-on-github)
Your [GitHub organization](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/about-organizations) can use Tailscale for free if you're using it for an open source
project which has an [OSI license](https://opensource.org/licenses).
You cannot use the [Billing](https://login.tailscale.com/admin/settings/billing) page of the Tailscale admin console to sign up for the Community on GitHub plan.
Consider using this plan when you need the following:
* Access a shared build tool.
* Access an internal test server for a given platform.
* Share project resources with other developers.
To use this pricing plan, you must be using [GitHub for authentication](/docs/integrations/identity). For more information about this plan, contact [Tailscale Support](/contact/support?type=billing).
## [Promo codes](#promo-codes)
Promo codes can be applied when upgrading to a new paid plan, or you can apply one to your existing paid plan in the [Billing](https://login.tailscale.com/admin/settings/billing) page of the admin console.
There are several ways to obtain a Tailscale promo code for a discount when upgrading to a paid plan. We distribute promo codes at trade shows and conferences, as well as through our various marketing channels, including emails, YouTube videos, and advertisements.
## [Charities, not-for-profit organizations, and educational institutions](#charities-not-for-profit-organizations-and-educational-institutions)
Charities, not-for-profit organizations, and educational institutions receive a 50% discount off of list prices. First, choose a plan in the [Billing](https://login.tailscale.com/admin/settings/billing) page of the admin console, then contact [Tailscale Support](/contact/support?type=billing) to get this discount.
