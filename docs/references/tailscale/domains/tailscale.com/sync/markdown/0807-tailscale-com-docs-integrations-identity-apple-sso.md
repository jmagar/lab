# Setting up Apple to work with Tailscale
Last validated: Sep 26, 2025
**You should be able to use your Apple ID to log into Tailscale without additional configuration.**
To use Apple as your identity provider with Tailscale, select **Sign up with Apple** when [signing up for your Tailscale account](https://login.tailscale.com/start).
Unfortunately, we cannot migrate your tailnet from/to Apple as an identity provider.
