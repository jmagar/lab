# Third-party integrations for JIT access
Last validated: Nov 12, 2025
Use third-party integrations to provide just-in-time (JIT) access to your Tailscale network (known as a tailnet). Use these integrations to add, update, or remove group and user settings in your [tailnet policy file](/docs/features/tailnet-policy-file).
