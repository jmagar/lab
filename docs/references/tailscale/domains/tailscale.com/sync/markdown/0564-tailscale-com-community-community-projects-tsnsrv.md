# tsnsrv
tsnsrv includes a little go program that sets up a reverse proxy listening on your Tailnet (optionally with a funnel), forwarding requests to a service reachable from the machine running this program. This is directly and extremely inspired by the [amazing talk by Xe Iaso](https://tailscale.com/blog/tsup-tsnet) about the wonderful things one can do with [`tsnet`](https://pkg.go.dev/tailscale.com/tsnet).
