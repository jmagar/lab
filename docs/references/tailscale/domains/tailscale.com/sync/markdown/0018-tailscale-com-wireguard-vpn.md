## Tailscale elevates the WireGuard® VPN experience
### WireGuard®-as-a-service
Tailscale provides a managed control plane to simplify key management to establish connectivity and encrypted communications for users, devices, and services.
### Identity-aware network
Tailscale automates user and group provisioning to safeguard against unauthorized resource access via SCIM integrations with leading identity providers.
### Infrastructure agnostic
Tailscale operates in almost any environment. Securing connectivity between on-premises, hybrid, and multi-cloud. It runs on any OS, runtime, and even on embedded systems.
## Establish a zero-trust network with Tailscale & WireGuard®
### Enforce the principle of least privilege
Designate which resources are accessible to which roles and groups to create dynamic software-defined perimeters across infrastructures.
