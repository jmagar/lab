# Install Tailscale on macOS
Last validated: Jan 5, 2026
The current version of the Tailscale client requires macOS Monterey 12.0 or later.
1. Download and install the Tailscale client using one of the following options:
* [Standalone variant](https://pkgs.tailscale.com/stable/#macos) from Tailscale's package server (recommended).
* [Mac App Store](https://apps.apple.com/app/tailscale/id1475387142) variant. The Tailscale client is free to download, but downloading any app from the Mac App Store may require an Apple ID with a valid credit card attached.
* [Our open source](https://github.com/tailscale/tailscale/wiki/Tailscaled-on-macOS) variant from our GitHub repository.
* Follow the Tailscale onboarding flow, which will guide you to install the Tailscale VPN configuration.
* Sign up with [your SSO identity provider](/docs/integrations/identity).
For more information about the different variant options, refer to [Three ways to run Tailscale on macOS](/docs/concepts/macos-variants).
