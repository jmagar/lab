# Aperture by Tailscale
Last validated: Apr 9, 2026
Aperture by Tailscale is currently [in beta](/docs/reference/tailscale-release-stages#beta).
Aperture by Tailscale is a centralized AI gateway that secures, monitors, and routes LLM requests across your organization. Aperture uses [Tailscale's identity layer](/docs/concepts/tailscale-identity) to automatically authenticate users, eliminating the need to distribute API keys. It routes requests to upstream LLM providers such as OpenAI, Anthropic, and Google without requiring changes to existing tools or workflows.
