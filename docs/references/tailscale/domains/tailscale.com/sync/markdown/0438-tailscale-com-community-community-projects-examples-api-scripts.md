# Examples-API-Scripts
The scripts in this repository demonstrate how to interact with Tailscale’s management API.
These scripts are designed to be minimally functional, and will generally require some customization or combining to produce useful results. If you're planning to work with the Tailscale API, this is a good starting point to understand how to do so.
