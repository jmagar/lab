# Tailscale now in the QNAP App Center
An official Tailscale app has landed in the QNAP App Center, so now users of the company’s network-attached storage devices can download and install a Tailscale client with just a few clicks.
QNAP users can install the Tailscale package [directly from the QNAP App Center](https://www.qnap.com/en-us/app_center/?os=qts&amp;version=5.1.0&amp;kw=tailscale). Please note that Tailscale is currently only available for x86-64bit and arm-64bit models, not arm41 devices.
It also remains possible to install Tailscale on QNAP devices through our [QPKG builder scripts](https://github.com/tailscale/tailscale-qpkg) — our thanks to [@ivokub](https://github.com/ivokub/) for creating that project, and transferring it over to us for continued development! We’ll keep updating the software in the App Center, but because of the review process, newer versions of Tailscale may be available via those builder scripts.
For more information about using Tailscale with QNAP machines and other NAS devices, check out the documentation [available on our knowledge base](/kb/1273/qnap).
