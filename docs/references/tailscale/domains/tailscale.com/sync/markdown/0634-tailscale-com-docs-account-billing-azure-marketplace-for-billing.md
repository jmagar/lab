# Tailscale on Azure Marketplace
Last validated: Dec 20, 2024
You can set up Tailscale billing through the Microsoft Azure Marketplace.
## [Prerequisites](#prerequisites)
* You must have an Azure account.
* Your Azure Account ID must have a Billing ID with an active billing profile aligned to it.
## [Purchase Tailscale on Azure Marketplace](#purchase-tailscale-on-azure-marketplace)
To set up billing for Tailscale from the Azure Marketplace:
1. Go to the [**Tailscale Plans + Pricing**](https://azuremarketplace.microsoft.com/en/marketplace/apps/tailscale1616171838087.tailscale_flat_rate_pricing?tab=PlansAndPrice) page in Azure Marketplace.
2. Find the plan you want to buy and select **Get it now**.
3. Respond to the prompts, read the policies, and accept the agreement if you agree.
## [Cancel your Tailscale subscription on Azure Marketplace](#cancel-your-tailscale-subscription-on-azure-marketplace)
Canceling your Tailscale billing subscription on Azure Marketplace does not delete your Tailscale network (known as a tailnet). For information about deleting your tailnet, refer to [Delete your tailnet](/docs/account/delete-tailnet).
To remove your billing subscription from Azure Marketplace:
1. Go to the [**SaaS**](https://portal.azure.com/#view/HubsExtension/BrowseResourceBlade/resourceType/Microsoft.SaaS/resources) page in the [Azure portal](https://portal.azure.com/).
2. Select the name of your Tailscale subscription and select **Delete**. You specified the name when you made your subscription.
3. If you also want to delete your Tailscale network, refer to [Delete your tailnet](/docs/account/delete-tailnet).
## [Considerations](#considerations)
If you would like to discus Tailscale custom pricing or a private offer by using the Microsoft Azure Marketplace, email `partnerships@tailscale.com`.
