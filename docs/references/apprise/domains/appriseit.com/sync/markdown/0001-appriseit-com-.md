## Why Apprise
One notification library to rule them all.
* **Universal:** A single, intuitive URL format for many notification providers.
* **Rich Content:** Attachments and images supported where the upstream service permits.
* **Lightweight:** Designed for automation workflows with minimal overhead.
* **Responsive:** Amazing response times because all messages sent asynchronously.
* **Flexible:** Works as a library, a CLI tool, or a self-hosted API service.
