## Description
This PR aims to fix issue [#1487](https://github.com/Tautulli/Tautulli/issues/1487) and provide a place where we can work & discuss on this topic.
**This PR is a updated version of [#1503](https://github.com/Tautulli/Tautulli/pull/1503) based on the latest version of Tautulli.**
**UI fields needing a change/fixed:**
* Recently Added
* Current Activity
* Library Recently Added
* User Recently Watched
* TV Show History
* Library Media Info
* Users Table
* Info page breadcrumb / header
* Newsletter templates
* Notification parameters
* Custom Format via Settings
Please report any occurrences of this issue that aren't listed above.
