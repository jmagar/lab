I dislike logging in to Tautulli and seeing news items from 4+ years ago on the main settings page.
### What is your feature request?

I'm not concerned with "End of Python 2 Support" or I'm not concerned with the CVE's that affected 2.15.3 and below as I've updated to the latest version. A way to remove news items or archive to a separate page would be fantastic.
