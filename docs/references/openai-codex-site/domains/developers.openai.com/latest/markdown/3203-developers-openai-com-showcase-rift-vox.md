Description
This browser FPS game was generated with Codex + GPT-5.5, with GPT Image 2 used for visual assets. It drops players into a neon arena with locked-down megablock atmosphere, weapon pickups, and fast first-person combat.

###
Iterations
1. Built the first voxel cyberpunk FPS prototype with arcade combat and a neon visual direction.
2. Improved spatial readability so the level felt like a real building with clearer walls, corridors, routes, and navigation.
3. Clarified enemy identity and visibility so players could understand what to track, fight, collect, and avoid.
4. Expanded the map into a larger explorable megablock with enemies, cores, movement space, and an extraction goal.
5. Pushed the game feel with stronger combat feedback, HUD states, lighting, materials, and more dramatic cyberpunk presentation.
6. Used GPT Image 2 for a cyberpunk voxel megablock start-screen image and integrated it into the entry flow.
7.
Final step
Finished the Neon FPS with a larger readable megablock level, clearer enemies, stronger cyberpunk combat feedback, and generated start-screen artwork.
