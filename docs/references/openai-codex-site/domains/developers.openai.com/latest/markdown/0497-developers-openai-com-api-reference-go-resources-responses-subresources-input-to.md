# Get input token counts
client.Responses.InputTokens.Count(ctx, body) (\*[InputTokenCountResponse](</api/reference/go/resources/responses#(resource) responses.input_tokens > (model) InputTokenCountResponse > (schema)>), error)
POST/responses/input\_tokens
Returns input token counts of the request.
Returns an object with `object` set to `response.input\_tokens` and an `input\_tokens` count.
