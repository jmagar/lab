## Get started
The current submission flow is still the path to public
distribution. When you publish an approved app, its available in
ChatGPT apps store and OpenAI creates a plugin for Codex
distribution. For now, plugins are only available in Codex.
Self-serve plugin publishing is coming soon. Learn more in the
[
App submission guidelines
](/apps-sdk/app-submission-guidelines)
and the
[
build plugins guide
](/codex/plugins/build).
