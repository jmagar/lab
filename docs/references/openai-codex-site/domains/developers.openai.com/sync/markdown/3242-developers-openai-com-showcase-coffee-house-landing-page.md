Description
This landing page was generated with Codex + GPT-5.5, using GPT Image 2 for visual assets. It presents a cold brew house with a quiet editorial layout, product callouts, and lifestyle imagery.
##
Build notes
###
Initial prompt
Copy prompt
```
Generate a landing page for a coffee house called Cold Brew House. First generate a beautiful UI mockup of this landing page, and also generate visual assets like a flat, colorful illustration representing coffee brewing. The landing page should be minimalist, modern, with an original touch.
```

###
Iterations
1. Created the initial Cold Brew House mockup with a minimalist modern coffee-house direction and generated coffee-brewing visuals.
2. Built the first static landing page with hero, signature, process, menu, visit, and footer sections.
3. Iterated toward the mockup by tightening the hero, improving the signature section, adding generated coffee photography, and refining spacing.
4. Reordered the page flow so the experience moved from hero to signature, process, menu, and visit.
5. Removed scroll-linked reveal effects so the page behaved like a clean standard landing page.
6. Fixed mobile and layout issues in the process, menu, and visit sections, including alignment, image sizing, anchors, and distracting decoration.
7.
Final step
Finished the Cold Brew House landing page with the final section order, generated imagery, normal scrolling, and cleaned-up mobile/menu/visit sections.
