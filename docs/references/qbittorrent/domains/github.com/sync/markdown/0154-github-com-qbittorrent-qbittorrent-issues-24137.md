### What is the problem?
qbt completely freezes up my m3 pro MacBook whenever I have multiple torrents (5+) running. The only way to fix this is with a full reboot.
### Steps to reproduce
Example:
Have 3 torrents seeding (they don’t even have to be actively uploading) and one large one (100gb+) downloading, when doing just those several, the app completely freezes up after a while. The only way to fix this is to completely reboot the system. Upon rebooting, qbt continues running, but vpn is turned off.
