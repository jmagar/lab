# Configuration
Tracearr can be configured through the web interface and environment variables.
## Web Interface[](#web-interface)
Most settings are available in the **Settings** page:
* **Servers** — Add and manage media server connections
* **Notifications** — Configure Discord webhooks and alerts
* **Sharing Detection** — Set up rules for account sharing detection
* **Appearance** — Customize the dashboard
* **[Tailscale VPN](/configuration/tailscale)** — Connect to your tailnet with the built-in Tailscale integration
## Environment Variables[](#environment-variables)
For advanced configuration, see the [Environment Variables](/configuration/environment) reference.
