Aurora / Type Study

Beautiful alternatives in the Noto Sans neighborhood

These are all clean, modern sans families that can carry Aurora’s premium control-plane tone without making the UI feel severe. The layout and color system stay fixed here; only the display typography changes.

Option A

Noto Sans

Balanced, international, neutral-beautiful. Very safe and very usable.

Noto Sans feels composed and human. It doesn’t try too hard, which is a strength if you want the product to feel accessible.
