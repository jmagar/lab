Labby Dark Theme Directions

Three premium control-plane directions using your blue-cyan accent range. All keep the current layout language but shift the dark palette for the rest of Labby. My recommendation is Aurora: it feels polished and modern without looking cold or intimidating.

Recommended
Aurora Control Plane

Soft navy-teal base with brighter cyan accents. Premium and modern, but still approachable because the contrast feels calm rather than severe.
